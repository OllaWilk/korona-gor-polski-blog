# ⛰️ "korona-gor-polski-blog": javaScript, handlebars
The "korona-gor-polski-blog" is a dynamic web application dedicated to hiking enthusiasts. It utilizes JavaScript and the Handlebars template engine to create an interactive experience for users interested in conquering the Crown of Polish Mountains—a challenge that involves reaching the summit of specific mountains in Poland.

Page consists of: 

##### a header,

##### a left column,
where the titles of articles will be located. A list of article titles is dynamically generated based on the articles present in the HTML code. This list allows users to switch between articles.

##### a central column,
which displays one article. Clicking on an article title displays the content of that article in the main view area.

##### a right column, 
which contains a tag cloud and a list of authors. Tags and authors are dynamically generated based on the articles' metadata. Users can filter articles by clicking on a tag or an author name.

## ⭐ Key Features:

1.Dynamic Article Display
  - only articles will be found in the HTML code,
  - upon opening the page, a list of titles, a tag cloud, and a list of authors will be generated using JS,
  - clicking on any of the titles in the left column will display the corresponding article in the central column,
  - clicking on a tag or author will filter the list of titles in the left column.
    
2.Responsive Design: Utilizing Bootstrap and custom SCSS, the blog is fully responsive and adapts to various screen sizes.

3. Interactive Elements: Users can interact with the blog by clicking on article titles, tags, and author names to filter the content displayed.
